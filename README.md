# Lazy Acres RV Park — Website

A single-page static website for Lazy Acres RV Park, Lenoir City, East Tennessee.

## Files
- `index.html` — the website (rename `lazy-acres-rv-park.html` to `index.html` for GitHub Pages)
- `photos/` — images used in the gallery

## What's on the site
Hero, the park, photo gallery, 2026 rates (back-in & pull-thru), nightly/weekly booking estimator,
monthly stay request, full guest info (park rules, site standards, seasonal notices), and contact.

> The booking and request forms are **demo-only** — they validate and confirm on screen but do not
> send or store anything, and never collect payment. Wire them to a form handler (e.g. Formspree or
> Netlify Forms) to make them live.

## Deploy with GitHub Pages
1. Create a new repository on GitHub (e.g. `lazy-acres-rv-park`), **Public**.
2. Upload `index.html` and the `photos/` folder to the `main` branch.
3. Go to **Settings → Pages**.
4. Set **Source: Deploy from a branch**, **Branch: `main` / `(root)`**, then **Save**.
5. The site goes live at `https://<your-username>.github.io/lazy-acres-rv-park/` within a minute or two.
